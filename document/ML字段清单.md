# ML字段清单

本清单对应当前 `src/Gurobi_Solver` 的输出口径。

推荐的机器学习任务定义：
- 预测标签使用 `full_runtime`
- `求解时间` 仅作为兼容旧实验的 legacy 口径
- 不将任何求解后结果列作为模型输入特征

## 不可用于 ML 输入

以下列不应作为输入特征：

- `序号`
- `求解选择`
- `random_seed`
- `gap`
- `总性价比`
- `总任务开始时间和`
- `总利润`
- `函数目标值`
- `求解时间`
- `legacy_runtime`
- `full_runtime`

说明：
- `序号` 只是实例内排序编号
- `求解选择` 与 `gap` 当前基本是常数或实验控制位，不提供稳定信息量
- `总性价比`、`总任务开始时间和`、`总利润`、`函数目标值` 都是求解后结果，会造成标签泄漏
- `求解时间` 是 legacy runtime，不是完整求解时间
- `legacy_runtime` 与 `full_runtime` 都属于标签侧信息，不应进入输入

## 推荐标签

- 主标签：`full_runtime`
- 辅助分析标签：`legacy_runtime`
- 兼容旧脚本标签：`求解时间`
