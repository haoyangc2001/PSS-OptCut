# PSS-OptCut

## 项目摘要
本项目面向产品-服务系统（PSS）的**配置与交付联合优化**问题，将“服务等级选择 + 团队路径排程 + 时间窗约束”统一建模为混合整数规划（MILP），在库存、工时、路线连续性等硬约束下同时优化客户效用、服务及时性与企业利润。项目围绕**有效不等式的定制与分阶段添加**展开求解效率对比，并系统记录实例特征与求解表现，形成数据集用于后续**机器学习预测求解时间/策略选择**。

## 核心创新点
- **有效不等式定制与分阶段添加**：构建 valid cut 池，并在“全局/根节点/回调节点/两阶段”不同方式下添加，对比求解效率与质量。
- **数据驱动预测**：批量生成实例、记录参数特征和求解时间，形成数据集支持策略选择与时效性预测。

## 优化模型（概要）
> 详细约束与变量实现见 `src/Solver_builder.py`。

### 决策变量
- `R[i,m]`：客户 `i` 是否选择服务等级 `m`。
- `X[i,j,k]`：团队 `k` 是否从节点 `i` 到 `j`。
- `Y[i,k]`：团队 `k` 是否服务客户 `i`。
- `Z[k]`：团队 `k` 是否出勤。
- `T[i,k]`：团队 `k` 在节点 `i` 的服务开始时间。

### 目标函数（加权多目标）
- 目标 1：客户效用/价格表现（性价比）
- 目标 2：服务开始时间（及时性）
- 目标 3：利润（收入 - 成本）
- 目标 4：回程完成时间（辅助约束项）

在 `src/main.py` 中通过 `Obj_Coef` 配置权重。

### 关键约束
- **服务选择**：每客户最多选择一种服务等级。
- **库存约束**：服务消耗产品不得超过库存。
- **工时约束**：团队总服务时长不超上限。
- **路径流平衡**：出发/返回与中间节点流守恒。
- **路径连续性**：转移时间 + 服务时间与开始时间一致。
- **时间窗**：服务开始时间落在客户最早/最晚时间窗内。
- **变量合理性**：服务选择、团队出勤与路径决策一致。

## 有效不等式（valid cut）设计
在 `src/main.py` 与 `src/Solver_builder.py` 中定义并应用，核心包括：

1. **反对称路径 cut**：
   `X[i,j,k] + X[j,i,k] <= 1`
2. **时间窗 cut（服务时长 + 转移时间）**：
   `T[i,k] + sum(R[i,m]*ServiceTime[m]) + Transfer[i][j] - BigM*(1 - X[i,j,k]) <= LateStart[j]`
3. **时间窗 cut（最小服务时长近似）**：
   `EarlyStart[i] + Transfer[i][j] + min(ServiceTime) - BigM*(1 - X[i,j,k]) <= LateStart[j]`

### 分阶段添加策略（示意）
- **全局添加（trigger1）**：求解前统一加入全部 cut。
- **根节点添加（trigger2）**：仅在根节点添加并求解，再移除。
- **回调添加（trigger3）**：在 MIPNODE 回调中按条件添加。
- **两阶段/混合策略（trigger4）**：根节点 + 回调组合使用。

## 预测设计（数据驱动）
- **数据生成**：`src/main.py` 随机生成多个实例，遍历 valid cut 组合并求解。
- **特征记录**：实例规模、参数分布统计（均值/标准差/分位数）、cut 组合、求解时间等。
- **输出数据集**：默认追加写入 `D:/桌面/result.csv`（需按本机路径修改）。
- **建模入口**：`notebooks/整体流程_预测时间 .ipynb`（可用于训练预测模型/策略选择）。

## 快速开始
### 1) 安装依赖
```bash
python -m pip install gurobipy numpy pandas openpyxl
```
> 需配置有效的 Gurobi 许可证。

### 2) 运行完整流程
```bash
python src/main.py
```
- 默认输出写入 `D:/桌面/result.csv`（Windows 路径），建议在 `src/main.py` 中改为本机路径，如 `data/result.csv`。

### 3) 进行预测/分析
```bash
jupyter notebook
```
打开 `notebooks/整体流程_预测时间 .ipynb`，加载生成的数据进行建模与评估。

## 常见调整点
- **输出路径**：修改 `src/main.py` 中 `df_target.to_csv(...)` 路径。
- **目标权重**：调整 `src/main.py` 的 `Obj_Coef`。
- **样本规模**：调整 `times` 循环次数与随机范围。
- **cut 组合策略**：修改 `src/sample_generator.py` 中的组合生成逻辑或回调触发条件。
